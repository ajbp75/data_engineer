# Raw E-Commerce Sales Dataset (Kaggle)

## Source
This dataset was obtained from Kaggle:  
https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data :contentReference[oaicite:2]{index=2}

## Purpose
Provide a broad view of e-commerce operations with multiple related data files, enabling analysis of orders, products, customers, payments and logistics.

## Layers
- **Raw:** original structured CSV files without transformation.
- **Bronze:** cleansed and validated events.
- **Silver:** semantically organized, analytics-ready.

## Characteristics of Raw Data
- Data may contain nulls, duplicates and inconsistent formats.
- Date and numeric fields may need transformation.
